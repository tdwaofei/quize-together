### 考点 1.1 标识符命名规则
**题目：** 以下关于 C++ 标识符命名规则的说法，正确的是：

A. 标识符可以以数字开头，如 `2ndValue`

B. `class` 可以作为变量名使用

C. 标识符只能由字母、数字和下划线组成

D. 标识符区分大小写，`Sum` 和 `sum` 是同一个变量

<details>
<summary>点击查看答案与解析</summary>

**答案：C**

**解题过程：**
- A：错误。C++ 标识符**不能以数字开头**。
- B：错误。`class` 是 C++ 的**关键字**（保留字），不能作为标识符使用。
- C：正确。标识符只能由字母（a-z, A-Z）、数字（0-9）和下划线（_）组成。
- D：错误。C++ 是**区分大小写**的语言，`Sum` 和 `sum` 是两个不同的标识符。

**本题考点：** C++ 标识符的命名规则与关键字。

**复习要点：**
- 标识符命名三要素：只能含字母/数字/下划线、不能以数字开头、不能是关键字。
- 常见关键字：`int`、`char`、`if`、`else`、`for`、`while`、`class`、`return` 等。
- 建议采用有意义的命名，如 `studentName` 或 `student_name`。
</details>

### 考点 1.2 数据类型转换
**题目：** 若有定义 `int a = 7; double b = 2.5;`，则表达式 `a / b` 的值是：

A. 2

B. 2.8

C. 3

D. 3.5

<details>
<summary>点击查看答案与解析</summary>

**答案：B**

**解题过程：**
- 在 C++ 中，当运算符两侧操作数类型不同时，会发生**类型转换**（向精度高的类型转换）。
- `a` 是 `int` 类型，`b` 是 `double` 类型，`a` 会被转换为 `double` 类型（7.0）。
- 所以 `a / b = 7.0 / 2.5 = 2.8`。
- 注意：如果写成 `a / 2`（两个 int 相除），结果才是整数 3。

**本题考点：** C++ 数据类型的自动转换规则。

**复习要点：**
- 整数相除结果还是整数（向零取整）：`7 / 2 = 3`，`-7 / 2 = -3`。
- 只要有一个操作数是浮点数，结果就是浮点数。
- 强制类型转换：`(double)a / b` 或 `double(a) / b`。
- 常见数据类型精度从低到高：`char` < `short` < `int` < `long long` < `float` < `double`。
</details>

### 考点 1.3 for循环
**题目：** 以下关于 `for` 循环的说法，正确的是：

A. `for` 循环的三个部分（初始化、条件、迭代）都必须填写

B. `for(;;)` 是语法错误的写法

C. `for` 循环的条件部分如果省略，默认为 `false`

D. `for(int i = 0; i < 10; i++)` 中，`i` 的作用域只在循环体内

<details>
<summary>点击查看答案与解析</summary>

**答案：D**

**解题过程：**
- A：错误。`for` 循环的三个部分都可以省略，如 `for(;;)` 表示无限循环。
- B：错误。`for(;;)` 是合法的无限循环，等价于 `while(true)`。
- C：错误。条件部分如果省略，默认为 `true`（无限循环）。
- D：正确。在 C++11 及以后标准中，循环变量 `i` 的作用域被限制在循环体内。

**本题考点：** `for` 循环的语法与变量作用域。

**复习要点：**
- `for(初始化; 条件; 迭代)` 的执行顺序：初始化 → 条件判断 → 循环体 → 迭代 → 条件判断...
- `break` 立即终止循环，`continue` 跳过当前迭代继续下一次。
- 注意区分 `i++` 和 `++i` 在 `for` 循环迭代部分效果相同。
</details>

### 考点 4.1 进制转换
**题目：** 十进制数 `100` 转换为二进制数是：

A. `1100100`

B. `1100110`

C. `1101000`

D. `1010000`

<details>
<summary>点击查看答案与解析</summary>

**答案：A**

**解题过程：**
使用"除 2 取余，逆序排列"的方法：

```
100 ÷ 2 = 50 ... 0
 50 ÷ 2 = 25 ... 0
 25 ÷ 2 = 12 ... 1
 12 ÷ 2 =  6 ... 0
  6 ÷ 2 =  3 ... 0
  3 ÷ 2 =  1 ... 1
  1 ÷ 2 =  0 ... 1
```

从下往上读取余数：`1100100`

验证：$1×2^6 + 1×2^5 + 0×2^4 + 0×2^3 + 1×2^2 + 0×2^1 + 0×2^0 = 64 + 32 + 4 = 100$ ✓

**本题考点：** 十进制转二进制的方法。

**复习要点：**
- 十进制转二进制：除 2 取余，逆序排列。
- 二进制转十进制：按权展开求和。
- 熟记 2 的幂次：$2^{10} = 1024$（1K），$2^{20} = 1048576$（1M）。
</details>

### 考点 2.2 队列
**题目：** 一个队列的入队顺序为 `1, 2, 3, 4, 5`，以下哪个**不可能**是出队顺序？

A. `1, 2, 3, 4, 5`

B. `5, 4, 3, 2, 1`

C. `1, 3, 2, 4, 5`

D. `1, 2, 4, 3, 5`

<details>
<summary>点击查看答案与解析</summary>

**答案：B**

**解题过程：**
队列是**先进先出**（FIFO）的数据结构。

- A：`1, 2, 3, 4, 5` — 每入队一个就立即出队，可以实现。
- B：`5, 4, 3, 2, 1` — 这是**栈**的出栈顺序（后进先出），队列无法实现。
- C：`1, 3, 2, 4, 5` — 可以实现。
- D：`1, 2, 4, 3, 5` — 可以实现。

**本题考点：** 队列的先进先出（FIFO）特性。

**复习要点：**
- 队列：先进先出（FIFO），入队在队尾，出队在队头。
- 栈：后进先出（LIFO），入栈和出栈都在栈顶。
- C++ STL：`queue<int> q;`，操作：`push()`、`pop()`、`front()`。
</details>

### 考点 9.4 递归函数
**题目：** 以下关于递归函数的说法，错误的是：

A. 递归函数必须包含终止条件（基准情况）

B. 递归函数的每次调用都会在内存中创建新的函数栈帧

C. 递归一定比循环效率高

D. 递归过深可能导致栈溢出

<details>
<summary>点击查看答案与解析</summary>

**答案：C**

**解题过程：**

**本题考点：** 递归函数的基本概念与特性。

**复习要点：**

- 递归 vs 循环：递归代码更简洁，但循环通常效率更高。
- 递归优化：记忆化搜索可以避免重复计算。
</details>

### 考点 4.3 冒泡排序
**题目：** 对数组 `[5, 3, 8, 1, 2]` 进行**冒泡排序**（从小到大），第一趟排序后的结果是：

A. `[3, 5, 1, 2, 8]`

B. `[1, 3, 5, 2, 8]`

C. `[3, 5, 1, 8, 2]`

D. `[5, 3, 1, 2, 8]`

<details>
<summary>点击查看答案与解析</summary>

**答案：A**

**解题过程：**
冒泡排序：相邻元素两两比较，逆序则交换，每一趟将最大的元素"冒泡"到末尾。

初始：`[5, 3, 8, 1, 2]`

第一趟比较过程：
- 比较 5 和 3：5 > 3，交换 → `[3, 5, 8, 1, 2]`
- 比较 5 和 8：5 < 8，不交换 → `[3, 5, 8, 1, 2]`
- 比较 8 和 1：8 > 1，交换 → `[3, 5, 1, 8, 2]`
- 比较 8 和 2：8 > 2，交换 → `[3, 5, 1, 2, 8]`

第一趟结果：`[3, 5, 1, 2, 8]`（最大的 8 已到位）

**本题考点：** 冒泡排序的基本原理。

**复习要点：**
- 冒泡排序：时间复杂度 O(n²)。
- 选择排序：每轮选最小放到已排序部分末尾。
- 插入排序：将元素插入到已排序部分的正确位置。
- C++ STL：`sort(a, a+n)`，时间复杂度 O(n log n)。
</details>

### 考点 2.5 二叉树遍历
**题目：** 已知二叉树的前序遍历为 `A, B, D, E, C`，中序遍历为 `D, B, E, A, C`，则该二叉树的后序遍历是：

A. `D, E, B, C, A`

B. `E, D, B, C, A`

C. `D, B, E, C, A`

D. `D, E, C, B, A`

<details>
<summary>点击查看答案与解析</summary>

**答案：A**

**解题过程：**
利用前序和中序重建二叉树：

1. 前序第一个元素 `A` 是根节点。
2. 在中序中找到 `A`，左边 `D, B, E` 是左子树，右边 `C` 是右子树。
3. 左子树前序为 `B, D, E`，中序为 `D, B, E`：
   - `B` 是左子树的根，`D` 是左孩子，`E` 是右孩子
4. 右子树只有 `C`

树结构：
```
      A
     / \
    B   C
   / \
  D   E
```

后序遍历（左右根）：`D, E, B, C, A`

**本题考点：** 由前序、中序遍历重建二叉树，并求后序遍历。

**复习要点：**
- 前序遍历：根 → 左 → 右
- 中序遍历：左 → 根 → 右
- 后序遍历：左 → 右 → 根
</details>

### 考点 5.1 DFS与BFS
**题目：** 以下关于深度优先搜索（DFS）和广度优先搜索（BFS）的说法，正确的是：

A. DFS 通常使用队列实现

B. BFS 通常使用栈实现

C. DFS 更适合寻找最短路径

D. BFS 在无权图中可以找到从起点到所有可达点的最短路径

<details>
<summary>点击查看答案与解析</summary>

**答案：D**

**解题过程：**
- A：错误。DFS 通常使用**栈**实现。
- B：错误。BFS 通常使用**队列**实现。
- C：错误。DFS 不保证找到最短路径。
- D：正确。BFS 按层扩展，第一次到达某点时的路径就是最短的。

**本题考点：** 深度优先搜索与广度优先搜索的实现与特性。

**复习要点：**
- DFS（深度优先）：使用栈，一条路走到底再回溯。
- BFS（广度优先）：使用队列，层层扩展，适合找最短路径。
- 时间复杂度：都是 O(V + E)。
</details>

### 考点 4.1 整数运算
**题目：** 已知 `int a = 17, b = 5;`，以下表达式的值等于 3 的是：

A. `a / b`

B. `a % b`

C. `a - b * 3`

D. `(a + b) / 10`

<details>
<summary>点击查看答案与解析</summary>

**答案：A**

**解题过程：**
- A：`a / b = 17 / 5 = 3`（整数除法向零取整）✓
- B：`a % b = 17 % 5 = 2`
- C：`a - b * 3 = 17 - 15 = 2`
- D：`(a + b) / 10 = 22 / 10 = 2`

**本题考点：** 整数除法与取模运算。

**复习要点：**
- `/` 是整数除法（两个整数相除时结果向零取整）。
- `%` 是取余运算。
- 整数运算公式：`a = (a / b) * b + a % b`。
</details>
